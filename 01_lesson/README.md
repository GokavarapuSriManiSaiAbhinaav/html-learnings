### Concepts Learned:

- `<!DOCTYPE html>`  
  It is not a tag. It tells the browser that the document is HTML5.  
  It must be written at the top of every HTML file.

- `<html lang="en">`  
  The `lang` attribute specifies the language of the document.  
  It helps browsers and search engines understand the content language.

- `<meta charset="UTF-8">`  
  It tells the browser how to read and display text correctly.  
  UTF-8 supports almost all languages and symbols.

- `<title>`  
  Sets the title of the webpage (shown in the browser tab).

- `<h1>` and `<p>`  
  Used for headings and paragraphs.
