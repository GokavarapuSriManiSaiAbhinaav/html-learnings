# Day 2 - HTML Head Elements

## Overview
Today I learned about elements inside the `<head>` section of an HTML document.  
The `<head>` section contains information about the webpage that helps browsers and search engines understand the page.

These elements are not directly visible on the webpage.

## Concepts Learned

---

### 1. Author Meta Tag
<meta name="author" content="Abhi">

This meta tag specifies the author of the webpage.

---

### 2. Description Meta Tag
<meta name="description" content="This page contains all the things I am learning as I learn HTML.">

This provides a short description of the webpage.  
Search engines may use this description when displaying search results.

---

### 3. Adding a Favicon
<link rel="icon" href="ad-sparkle-image.png" type="image/x-icon">


This adds a small icon (favicon) to the browser tab.

- `rel="icon"` specifies the relationship as a website icon.
- `href` defines the path of the icon file.
- `type` specifies the file type.
