# Day 3 - HTML Text Basics

## What I Learned

Today I learned basic text formatting tags in HTML.

### Headings
I learned how to use heading tags like:
<h1> to <h3>

These tags are used to give titles and subtitles to the content.

### Line Break
<br> tag is used to move text to the next line.

### Text Formatting
- <strong> is used to show important text (bold meaning).
- <em> is used to emphasize text (italic meaning).

### Special Characters
I learned how to display special symbols using HTML entities:
- &gt  → Greater than symbol
- &lt  → Less than symbol
- &copy → Copyright symbol
- &nbsp → Adds extra space

### Abbreviation Tag
<abbr> tag is used to show short forms.
When we hover on it, full meaning can be shown.

## Summary
In this lesson I understood how to structure and format text properly in a webpage.
